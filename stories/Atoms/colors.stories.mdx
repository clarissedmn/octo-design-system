# 🌈 Colors

#

#

#### Brand colors define the look of our websites but also our values. In this way, they have to be carefully use.

#

---

- To apply a design system color in your **_SCSS file_**, use a **SCSS variable**.

  **_Exemple :_**

      .my-class {

          color : $primary-dark;

      }

- To apply a design system color in your **_JS file_** :

  1.  Import the **_SCSS file_** :

         import { colors } from "octods";

  2.  Use camelCase :

      **_Exemple :_**

           colors.primaryDark
