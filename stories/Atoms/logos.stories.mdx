#### OCTO Technology is represented by two logos. A corporate one, used most of the time and a coded logo that will appear rarely.

#

To use a logo,

1. Copy the **name** of SvgComponent
2. Define its `width` or/and `height` props with a number (pixels)
3. Specify `white` props on dark background
4. â™¿ Add information in `text` that describe the function of the logo

**_Exemple :_**

- Colors Logo :

```
    <SvgComponent width={number} height={number} text="..."/>

```

- Monochrome Logo :

```
    <SvgComponent width={number} height={number} text="..." color="white"/>
```
