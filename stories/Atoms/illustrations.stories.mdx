# 🎨 Illustrations

#

#

#### Brand colors define the look of our websites but also our values. In this way, they have to be carefully use.

#

---

To use an illustration,

1. Copy the **name** of SvgComponent
2. Define its `width` or/and `height` props with a number (pixels)
3. In `color`, add a [SCSS variable color](/?path=/docs/⚛%EF%B8%8Fatoms-🌈-colors--page)
4. ♿Specify if the illustration is :

- Descriptive :

```

  <SvgComponent width={number} height={number} color="..."/>

```

- Informative (add information in `title` that describe the illustration) :

```

  <SvgComponent title="..." width={number} height={number} color="..." />

```
